Workflow Builder Test

To see the vertical workflow builder:

  1. Start your development server: npm run dev
  2. Navigate to the Automations page: /automations
  3. Click on the "Workflows" tab (3rd tab)
  4. Click "Create Workflow" button

Features:

  • ✅ Vertical drag-and-drop interface
  • ✅ Inline editing (no side panel)
  • ✅ Expandable/collapsible steps
  • ✅ Visual connection lines
  • ✅ Enable/disable steps
  • ✅ Duplicate & delete actions
  • ✅ Multiple trigger types
  • ✅ Various action types
  • ✅ Mobile responsive

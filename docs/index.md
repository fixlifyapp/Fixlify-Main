## Project Summary

**Fixlify** is a comprehensive repair shop management SaaS platform built with:

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Supabase (PostgreSQL, Edge Functions, Auth, Realtime)
- **Integrations**: Telnyx (VoIP/SMS), Mailgun (Email), OpenAI (AI)
- **Deployment**: Vercel (Frontend) + Supabase Cloud

## Core Features

### Business Operations
- **Client Management** - Profiles, properties, contact history
- **Job Management** - Work orders, scheduling, status tracking
- **Financial** - Estimates, invoices, payments
- **Inventory** - Parts and supplies tracking
- **Team** - Multi-user with RBAC

### Communications
- **SMS** - Two-way messaging via Telnyx
- **Email** - Transactional emails via Mailgun
- **Voice** - VoIP calls with AI dispatcher
- **Automation** - Trigger-based workflows

### AI Features
- **AI Voice Dispatcher** - Automated call handling
- **AI Message Generation** - Smart message composition
- **AI Workflow Assistant** - Automation suggestions
- **AI Client Lookup** - Intelligent search
